# Jarvis-Your_Personal_AI_Assistant
Jarvis is an intelligent AI-powered assistant built using Python and Google Gemini API. It can understand natural language, answer questions, and interact like a real assistant.

Features
- Natural language conversation
- Powered by Gemini 2.0 Flash / Pro
- Memory-based conversation using chat sessions
- Clean and modular Python code
- Fast responses and lightweight
- Easy to modify and expand (add voice, tools, automation, etc.)
